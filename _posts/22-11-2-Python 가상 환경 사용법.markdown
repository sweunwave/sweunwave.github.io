Python 가상환경 만들기

```python -m venv 가상환경이름```

가상환경 활성화

```가상환경이름\Scripts\activate.bat```

비활성화

```deactivate```

가상환경에서 패키지 설치할땐 그냥 pip install 로 하면됨

패키지목록관리

```pip freeze```

가상환경 삭제하고싶으면 그냥 휴지통에 넣으면 된다.
